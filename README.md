# PM2.5 環境空氣顯示器 (Airbox 、 LASS 等民間Maker站點)
* 顯示由「PM2.5開放資料入口網站」接入的測站資料，全台灣約3000+個民間站點。
* 支援透過GPS定位、地址輸入定位，快速選擇最靠近你的測站。

使用App Inventor2撰寫\
數據資料暫存於/PM25Air/資料夾下，可自行清空
